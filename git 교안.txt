* 깃(Git)
1) 버전 관리 시스템
2) 백업 : 깃허브(gitHub) 사용
3) 협업 : 팀원들과 파일을 공유

1. 깃 환경 설정
2. 초기화
: 폴더 별로 초기화함.

3. 스테이징, 커밋하기
- 작업트리 : 파일수정, 저장등의 작업을 하는 디렉터리
- 스테이지 : 파일 대기 공간(<- 스테이징)
- 저장소 : 버전을 만들어 저장하는 곳(<- 커밋)
=> 커밋 후 새로운 버전이 생성되면서 스테이지에 대기하던 파일이 모두 저장소로 올라감.

(작업트리) -> (스테이지) -> (저장소)
작업하는 디렉터리 안에 .git 숨김파일로 존재함.
